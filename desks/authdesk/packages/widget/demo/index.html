AuthDesk 인증 위젯

아래 폼으로 end-user 를 가입/로그인시키세요. 성공하면 사용자·토큰이 콘솔에 찍히고 폼이 로그인 상태 카드로 바뀝니다. 설정을 바꾸면 즉시 다시 마운트됩니다.

로컬 API 가 떠 있어야 합니다. AUTHDESK_MODE=self-hosted pnpm --filter @authdesk/api run dev (기본 포트 4110, PGlite 폴백). 시드된 pk_demo 테넌트 · 데모 사용자 ada@demo.test / Password123! 로 로그인됩니다.

바닐라(비-React) 임베드

빌드한 IIFE 를 비-React 페이지에 붙일 때:

<div id="authdesk-form"></div>
<script src="https://auth.example.com/authdesk-widget.js"></script>
<script>
  AuthDesk.init({
    target: '#authdesk-form',
    publishableKey: 'pk_…',
    endpoint: 'https://auth.example.com',
  })
</script>
