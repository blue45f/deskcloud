RealtimeDesk Presence 위젯

아래 PresenceBar 는 채널에 접속한 사람들을 실시간 아바타·카운트로 보여줍니다. 다른 탭에서 같은 채널로 접속하면 카운트가 즉시 올라갑니다.

로컬 API 가 떠 있어야 합니다. pnpm --filter @realtimedesk/api run dev (기본 포트 4092, PGlite 폴백). 시드된 pk_demo 테넌트(corsOrigins ['*'])로 접속합니다.

바닐라(비-React) 임베드

빌드한 IIFE 를 비-React 페이지에 붙일 때: 

<div id="presence"></div>
<script src="https://realtime.example.com/realtime-widget.js"></script>
<script>
  RealtimeDesk.init({
    target: '#presence',
    channel: 'room:42',
    publishableKey: 'pk_...',
    endpoint: 'https://realtime.example.com',
  })
</script>
