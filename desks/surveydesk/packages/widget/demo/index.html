SurveyDesk 피드백 위젯

우하단의 피드백 버튼을 눌러 위젯을 확인하세요. 아래 설정을 바꾸면 다른 appId·엔드포인트·강조색·위치로 즉시 다시 마운트됩니다.

로컬 API 가 떠 있어야 합니다. pnpm --filter @surveydesk/api run start:dev (기본 포트 4090, PGlite 폴백). 시드된 demo 설문이 자동으로 보입니다.

바닐라(비-React) 임베드

빌드한 IIFE 를 비-React 페이지에 붙일 때: 

<script src="https://surveys.example.com/feedback-widget.js"></script>
<script>
  SurveyDesk.init({ appId: 'offhours', endpoint: 'https://surveys.example.com' })
</script>
