ModerationDesk 위젯

아래 댓글 옆 신고 버튼을 눌러 접근성 다이얼로그(사유 선택 + 상세)를 확인하세요. 작성칸에 입력하면 <ModerationBadge> 가 publishable 키로 실시간 사전검사 결과를 배지로 보여 줍니다.

로컬 API 가 떠 있어야 합니다. pnpm --filter @moderationdesk/api run start:dev (기본 포트 4092, PGlite 폴백). 시드된 demo 테넌트의 pk 를 넣으세요.

댓글 목록 + 작성칸 사전검사

각 댓글 옆 <ReportButton>, 맨 아래 작성칸에 <ModerationBadge>. 금칙 규칙(시드된 욕설/스팸 등)에 걸리면 배지가 뜹니다. 차단은 서버 몫.

바닐라(비-React) 임베드

빌드한 IIFE 를 비-React 페이지에 붙일 때:

<span id="report-slot"></span>
<script src="https://moderate.example.com/report-button.js"></script>
<script>
  ModerationDesk.init({
    target: '#report-slot',
    subjectType: 'comment', subjectId: 'c_123',
    publishableKey: 'pk_...', endpoint: 'https://moderate.example.com'
  })
</script>
