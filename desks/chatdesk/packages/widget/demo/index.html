ChatDesk 채팅 위젯

우하단의 채팅 버튼을 눌러 위젯을 확인하세요. 시드된 demo 테넌트의 DM·그룹 대화가 보입니다. memberId 를 alice / bob 로 바꿔 두 창을 띄우면 실시간으로 메시지가 오갑니다.

로컬 API 가 떠 있어야 합니다. PORT=4094 pnpm --filter @chatdesk/api run start:dev (PGlite 폴백, 시드 demo 테넌트 pk_demo/sk_demo). 멤버는 alice·bob·carol.

바닐라(비-React) 임베드

빌드한 IIFE 를 비-React 페이지에 붙일 때:

<script src="https://chat.example.com/chat-widget.js"></script>
<script>
  ChatDesk.init({
    publishableKey: 'pk_demo',
    endpoint: 'https://chat.example.com',
    memberId: 'alice',
  })
</script>
