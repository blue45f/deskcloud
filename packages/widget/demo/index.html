ChangelogDesk 위젯

우하단의 벨 버튼을 눌러 변경 이력 패널을 확인하세요. 미읽음이 있으면 배지가 표시되고, 패널을 열면 읽음 처리되어 배지가 사라집니다. 아래 설정을 바꾸면 다른 퍼블리시 키·엔드포인트·강조색·위치로 즉시 다시 마운트됩니다.

로컬 API 가 떠 있어야 합니다. pnpm --filter @changelogdesk/api run start:dev (기본 포트 4095, PGlite 폴백). 시드된 pk_demo 테넌트(corsOrigins '*')의 샘플 항목이 자동으로 보입니다.

바닐라(비-React) 임베드 — 외부 고객

빌드한 IIFE 를 비-React 페이지에 붙일 때:

<script src="https://changelog.example.com/changelog-widget.js"></script>
<script>
  ChangelogDesk.init({ publishableKey: 'pk_…', endpoint: 'https://changelog.example.com' })
</script>
