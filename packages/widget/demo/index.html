AdDesk 배너 위젯

아래 슬롯에 가중 선택된 활성 크리에이티브가 노출됩니다. 노출/클릭은 자동 추적되고, 결과가 콘솔에 찍힙니다. 설정을 바꾸면 즉시 다시 마운트됩니다.

로컬 API 가 떠 있어야 합니다. ADDESK_MODE=self-hosted pnpm --filter @addesk/api run dev (기본 포트 4096, PGlite 폴백). 시드된 pk_demo 테넌트로 서빙됩니다 (슬롯: sidebar, feed).

배너 슬롯
바닐라(비-React) 임베드

빌드한 IIFE 를 비-React 페이지에 붙일 때:

<div id="addesk-slot"></div>
<script src="https://ads.example.com/addesk-widget.js"></script>
<script>
  AdDesk.init({
    target: '#addesk-slot',
    slot: 'sidebar',
    publishableKey: 'pk_…',
    endpoint: 'https://ads.example.com',
  })
</script>
