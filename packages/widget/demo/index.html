MediaDesk 미디어 위젯

아래 publishableKey·endpoint·folder 를 바꾸면 업로더와 갤러리가 즉시 다시 마운트됩니다. 업로드가 성공하면 갤러리가 자동 새로고침됩니다.

로컬 API 가 떠 있어야 합니다. pnpm --filter @mediadesk/api run start:dev (기본 포트 4191, PGlite 폴백). 시드된 pk_demo 테넌트로 동작합니다.
