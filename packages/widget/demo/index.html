CommunityDesk 위젯

호스트 앱에 게시판/카페를 임베드하는 <CommunityBoard>(좌측)와 최근 글 요약 <CommunityFeed>(우측)입니다. 글을 열면 중첩 댓글이 보이고, memberId 를 주면 글/댓글 작성·반응이 켜집니다.

로컬 API 가 떠 있어야 합니다. pnpm --filter @communitydesk/api run start:dev (기본 포트 4096, PGlite 폴백). 시드된 pk_demo 테넌트의 free/notice 게시판, dev-cafe 카페가 자동으로 보입니다.

바닐라(비-React) 임베드 
<div id="community"></div>
<script src="https://community.example.com/community-widget.js"></script>
<script>
  CommunityDesk.init({
    target: '#community', boardSlug: 'free',
    publishableKey: 'pk_...', endpoint: 'https://community.example.com',
    memberId: 'u_42', memberName: '준호'
  })
</script>
