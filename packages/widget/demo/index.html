NotifyDesk 알림 벨 위젯

우상단 벨 아이콘을 눌러 인박스를 확인하세요. 미읽음이 있으면 배지가 뜨고(주기적 폴링), 패널을 열면 읽음 처리됩니다. 아래 설정을 바꾸면 즉시 다시 마운트됩니다.

로컬 API 가 떠 있어야 합니다. NOTIFYDESK_MODE=self-hosted pnpm --filter @notifydesk/api run dev (기본 포트 4095, PGlite 폴백). 시드된 pk_demo 테넌트 인박스가 보입니다.

바닐라(비-React) 임베드

빌드한 IIFE 를 비-React 페이지에 붙일 때:

<div id="notify-bell"></div>
<script src="https://notify.example.com/notify-widget.js"></script>
<script>
  NotifyDesk.init({
    target: '#notify-bell',
    recipientId: 'user_42',
    publishableKey: 'pk_…',
    endpoint: 'https://notify.example.com',
  })
</script>
